# API Turnos Médicos - Spring Boot

API REST para la gestión de turnos médicos desarrollada con:

- Java 17
- Spring Boot 3
- Spring Data JPA
- MySQL
- Swagger / OpenAPI
- DTO + Mapper
- Manejo global de excepciones
- Paginación y filtros

## Funcionalidades

- Crear turno
- Listar turnos con filtros
- Cancelar turno
- Validación de turnos duplicados
- Documentación interactiva con Swagger

## Swagger

http://localhost:8080/swagger-ui.html

## Base de datos

MySQL local
